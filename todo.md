# v1.0.0
使用 pyside 开发一个应用
配置一个目录默认文档目录
一个按钮，点击后扫描该目录下的所有的 docx, doc, txt, pdf 文件,将 这些文件的内容提取出来, 并使用本地向量数据库保存提取出来的内容,向量数据库文件可以保存在应用 data/db 目录下
应用界面应该有个查询区域
    有个 textarea 输入框, 由于输入查询文本
    有个展示数量的输入框, 用于输入展示的数量, 默认为 10
    有个查询按钮
        点击后, 会将输入框中的内容作为查询条件, 从向量数据库中提取出相关的内容, 并展示在结果展示区
        如果本地未存在向量数据库文件, 则先提示用户先点击扫描按钮, 扫描后再查询
        如果在目录扫描时间里, 按钮不可用
    有个结果展示区
        用于展示查询到匹配到文件地址和匹配到底区域
        点击可以打开对应的文件
打包成一个无需安装的应用， 可以在 windows, mac, linux 上运行,发布到 dist 目录下 并带上版本号
